# 主题更新说明

## ✨ 新特性

### 1. 极简设计风格
- 干净的配色方案（蓝紫色渐变头部）
- 充足的留白和清晰的视觉层次
- 卡片式布局，提升内容可读性

### 2. 分栏布局
- **左侧栏**：项目经历、技术栈
- **右侧栏**：文章入口、兴趣爱好、小游戏
- 响应式设计，移动端自动变为单栏

### 3. 按钮动画效果
- **悬停效果**：边框变色、背景渐变、向右平移
- **光效扫过**：鼠标悬停时有光线从左到右扫过
- **箭头动画**：悬停时箭头向右移动
- **点击波纹**：点击按钮产生水波纹扩散效果

### 4. 优化的访客统计
- 将 GitHub 统计和访客计数并排显示
- 访客计数器改为更简洁的样式（蓝色扁平风格）
- 添加悬停放大效果
- 加载失败时显示友好提示

### 5. 其他动效
- **淡入动画**：页面滚动时元素逐渐显现
- **卡片悬浮**：鼠标悬停卡片轻微上浮
- **标签交互**：技能标签悬停时变色放大
- **排行榜项**：悬停时向右滑动

## 🚀 使用方法

1. **推送到 GitHub**
   ```bash
   git add .
   git commit -m "feat: 更新为自定义极简主题"
   git push
   ```

2. **等待 GitHub Pages 部署**
   - 通常 1-2 分钟内完成
   - 访问 `https://Lineus.github.io` 查看效果

3. **本地预览（可选）**
   ```bash
   # 安装 Jekyll（如果未安装）
   gem install jekyll bundler
   
   # 启动本地服务器
   jekyll serve
   
   # 访问 http://localhost:4000
   ```

## 🎨 自定义选项

### 修改配色
在 `assets/css/style.css` 的 `:root` 部分修改：
```css
:root {
    --primary-color: #2563eb;      /* 主色调 */
    --secondary-color: #64748b;    /* 次要颜色 */
    --text-color: #1e293b;         /* 文字颜色 */
    --bg-color: #ffffff;           /* 背景色 */
}
```

### 调整动画速度
在 CSS 中搜索 `transition` 和 `animation` 属性修改时间值。

### 添加更多文章
在 `index.html` 的 `.article-list` 中添加新的 `.article-item`。

## 💡 技术亮点

4. **跨浏览器兼容**：使用标准 CSS 和 JavaScript API
